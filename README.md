This repo contains: 
1. a powershell script to read a folder of PDF files from Google cloud storage and save this in the /public folder of a React project
2. a React component to display a "search" button, search the /public json and display the results

The circumstances that led to the devlopment of this code are described at [Searching PDF files - GCSE and "local-code" options](https://dev.to/mjoycemilburn/searching-pdf-files-coding-a-goggle-custom-search-engine-gcse-component-in-react-36fk)
